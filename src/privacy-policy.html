Security:

  • All passwords are hashed using proprietary algorithm combining MD5 and password hashing algorithms
  • All hardware resides in a secure virtual private cloud with configured security groups blocking all low-level access
  • All code commits run through a GIT based repository to a secure and isolated network (VPC). This allows us to design a highly secure network and isolate many of the resources behind multiple layers of firewalls and gateways that prevented direct access to all but the load balancer
  • All changes are security tested in staging and testing environments
  • Periodic security checks
